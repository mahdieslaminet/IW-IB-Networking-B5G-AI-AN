# IW-IB-Networking-B5G-AI-AN
یک دیتاست جدید برای یک شبکه خودکار مبنی بر هوش مصنوعی برای شبکه ۵g و فراتر از آن ارائه داده اند که شبیه سازی بر مبنای شبکه های اینترنت اشیا و شبکه ۵g ساختگی است که در لایه های مختلف شبکه نتایج یک شبکه از حمله شبکه ، پهنا باند ، ترافیک و ... را نمایش می‌دهد.

**پروژه پایانی درس شبکه مخابراتی**

# دمو:

این پروژه یک رابط کاربری وب یا دسکتاپ است که برای تعامل با سامانه‌های موجود در پروژه اصلی طراحی شده. این رابط UI می‌تواند برای:
نمایش وضعیت شبکه‌های B5G
ارسال و دریافت داده‌ها از APIهای پروژه
کنترل و مشاهده نتایج مدل‌های هوش مصنوعی
و تعامل بصری با بخش‌های مختلف سیستم
مورد استفاده قرار گیرد.
